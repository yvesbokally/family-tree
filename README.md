# Arbre Généalogique - Famille BOKALLY BIKANDA

## 🌳 À propos
Arbre généalogique interactif de la famille BOKALLY BIKANDA, de l'ancêtre KWASIO jusqu'à nos jours.

## ✨ Fonctionnalités
- Visualisation interactive avec zoom/drag
- Ajout de nouveaux membres (frères, cousins, descendants)
- Export/Import des données en JSON
- Sauvegarde automatique dans le navigateur

## 👨‍👩‍👧‍👦 Comment ajouter un membre ?
1. Cliquez sur "Ajouter un membre"
2. Remplissez le nom et sélectionnez le père
3. Cliquez "Ajouter"

## 💾 Partager l'arbre
- Exporter les données → Envoyer le fichier JSON à un cousin
- Il peut importer le fichier pour voir l'arbre complet
